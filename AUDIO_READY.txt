WHEN EACH AUDIO PLAYS
================================================================================

welcome.mp3:
   Trigger: Click "Insert Card" button
   Condition: Security check passes (ACCESS GRANTED)
   Message: "Welcome to secure ATM transactions"

transaction_complete.mp3:
   Trigger: Withdrawal completed successfully
   Condition: Transaction processed
   Message: "Thank you, your transaction is successfully completed"

mask_detected.mp3:
   Trigger: Face mask detected covering nose/mouth
   Condition: Mask properly worn
   Message: "Face mask detected. Please remove mask..."

helmet_detected.mp3:
   Trigger: Helmet detected (face region or full frame)
   Condition: Helmet worn
   Message: "Helmet detected. Please remove helmet..."

mask_and_helmet.mp3:
   Trigger: Both mask AND helmet detected
   Condition: Wearing both coverings
   Message: "Mask and helmet detected. Remove all coverings..."

multiple_people.mp3:
   Trigger: 2 or more people in frame
   Condition: Multiple faces detected
   Message: "Multiple people detected. Only one person allowed..."

================================================================================
TROUBLESHOOTING
================================================================================

No Audio?
1. Check volume (not muted, >50%)
2. Check speakers/headphones connected
3. Run: python test_all_audio.py
4. Check pygame: pip install pygame-ce

Wrong Audio Playing?
1. Check violation type in console logs
2. Verify file names match exactly
3. Check audio/ directory contents

Audio Too Frequent?
1. Rate limit is 8 seconds (working correctly)
2. To change: Edit config.json → rate_limit_seconds
